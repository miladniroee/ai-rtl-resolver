# Ai RTL Resolver

A browser extension that fixes the text direction on Chatgpt and Deepseek.


![Extension Screenshot](extension.jpg)


## Installation

> [!WARNING]
> Extension on store is not available now due to Iran internet restrictions.<br>
Download zip and use `load unpacked`  option on browser. 

### Chrome / Edge / Brave / Opera / Firefox

1. Download the latest release: [ai-rtl-resolver)-v2.0.zip](https://github.com/miladniroee/ai-rtl-resolver/releases)
2. Unzip the file
3. Open your browser and go to:
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
   - **Brave**: `brave://extensions/`
   - **Opera**: `opera://extensions/`
   - **Firefix** `about:debugging#/runtime/this-firefox`
4. Enable **Developer mode** (not for firefox, toggle in top-right corner)
5. Click **Load unpacked**
6. Select the unzipped folder

<br><br>



> Firefox version is not available, use zip to load unpacked.
